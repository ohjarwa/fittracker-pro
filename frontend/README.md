# FitTracker Pro Frontend

智能健身训练管理系统前端应用

## 技术栈

- **框架**: Vue 3.5+ (Composition API + \`<script setup>\`)
- **语言**: TypeScript 5.9+
- **构建工具**: Vite 7.3+
- **样式**: Tailwind CSS 4.1+
- **状态管理**: Pinia 2.3+
- **路由**: Vue Router 4.5+
- **HTTP 客户端**: Axios 1.8+
- **图表库**: ECharts 5.6+
- **工具库**: VueUse, date-fns, zod

## 快速开始

### 安装依赖

\`\`\`bash
npm install
\`\`\`

### 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

访问 http://localhost:5173

### 构建生产版本

\`\`\`bash
npm run build
\`\`\`
